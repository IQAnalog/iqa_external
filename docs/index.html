Introduction

This F1000 Software Developer's Kit (SDK) consists of a compiled library of Application Programming Interface (API) functions, header files protyping those APIs, binary firmware to be downloaded to the chip, and integration guidelines for using the SDK.

Also include is a 'src' directory with source code for the F1000 client used to communicate with the F1000 EVM.

Since the hardware configuration may have multiple F1000s installed, the library uses a Hardware Abstraction Layer (HAL) to provide the proper hardware dependent interface to a specific F1000. Details of the HAL are implementation dependent ... an example HAL is provided in the 'src' directory which interfaces with a single F1000 device on the F1000 EVM.

Directory Organization

There are several directories in this release and are organized as follows: 

 binaries  RISC-V and Framer binary FW to be downloaded to the F1000
 doc       Documentation about the release, including this HTML
 examples  Examples on how many of the APIs are being called
 include   C include files with function prototypes and macros
 lib       Pre-compiled library file to be linked with the client's code
 src       Source code for several utilities and routines to manage SPI access through the Hardware Abstraction Layer (HAL)
