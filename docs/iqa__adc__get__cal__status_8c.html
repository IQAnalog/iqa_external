This API reads the calibration status of the specified ADC.

The ADC ID is a one-based number as specified in "iqa_f1000_api.h". The values in the uint32_t returned can be accessed using the macros defined in the iqa_f1000_chip.h header file.

Parameters
f1000_id: An Integer that specifies the F1000 ID - Default 1
ep_idValue that represents the Element Processor ID.
Element Processor Value
A 0x1
B 0x2
adc_idValue that represents the ADC to be status'ed
pStatusPointer to the status return value.
Returns
: IQA_SUCCESS on Success else ERROR
