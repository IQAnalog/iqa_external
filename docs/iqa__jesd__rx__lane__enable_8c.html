int  ep_id,

)

This API enables JESD204b RX lanes in the F1000 EP.

All lanes that are enabled will be part of the JESD link. Note that the RX Framer / transport layer should be initialized prior to enabling the RX lanes. This is accomplished by calling the iqa_merlin_load_firmware API.

Parameters
f1000_id: An Integer that specifies the F1000 ID - Default 1
ep_idValue that represents the Element Processor ID.
Element Processor Value
A 0x1
B 0x2
lane_maskA bit mask of lanes to be enabled. Bit 0 represents lane 1, Bit 1 represents lane 2, etc.
Returns
: IQA_SUCCESS on Success else ERROR
