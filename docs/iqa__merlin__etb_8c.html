int iqa_merlin_etb_get ( int  f1000_id,
int  ep_id,
int  line,
char *  buf,
int  len,
int  formatted 
)

This API retrieves data from the trace data buffer. There are 1024 lines of trace data available for retrieval. When formatted data is requested (formatted = 1), line 0 returns the heading string. Parameter 'line' goes from 1 to 1024 for actual trace data.

Parameters
f1000_id: An Integer that specifies the F1000 ID - Default 1
ep_id: Value that represents the Element Processor ID.
Element Processor Value
A 0x1
B 0x2
line: Line number of trace data to be retrieved
buf: Buffer to receive data
len: Length of buffer
formatted: 1 for formatted data to be returned; 0 for unformatted raw data
Returns
: IQA_SUCCESS on Success else negative error number.
