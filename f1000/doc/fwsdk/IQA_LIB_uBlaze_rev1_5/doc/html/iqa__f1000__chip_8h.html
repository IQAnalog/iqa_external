◆ iqa_chip_is_rev_a()

bool iqa_chip_is_rev_a ( int  f1000_id)

This API checks whether F1000 chip is revision A.

See also
iqa_init in iqa_f1000_embedded_controller.h
Parameters
f1000_id: An Integer that specifies the F1000 ID - Default 1
Returns
: returns true if F1000 chip is rev A, else false.

Note: This function handles if no F1000 is detected which is treated as no detection has been performed. So call f1000_chip_read_rev() to retrieve chip revision. Once valid chip revision is obtained (procfs entry esists), this function returns immediately without further checking or detection.

This API initializes the Analog Block which includes the ADC and DAC.

This API initializes the Analog Block, including the clock receiver, the LDOs, the ADCs and DACs. The ADCs and DACs can be independently enabled/disabled and the supporting blocks are configured accordingly. The ADCs' calibration modules are also started.

Attention
math library is required
Parameters
f1000_idint : The F1000 ID
ep_idint : A masked value that represents the Element Processor ID.
Element Processor Mask Value
A 0x1
B 0x2
adc_idint : A masked value that represents the ADC ID.
ADC Mask Value
NO ADC Init0x0
1 0x1
2 0x2
1&2 0x3
dac_idint : A masked value that represents the DAC ID.
DAC Mask Value
NO DAC Init0x0
1 0x1
2 0x2
1&2 0x3
fclkuint64_t : Clock Frequency in Hz
cbconst cb_dac_s* : cb is a pointer to cb_dac_s structure containing the callback functions to be provided by the caller. These callback functions will be invoked during the execution of the API.
Returns
int : IQA_SUCCESS if no error. Error value if error occurs.
