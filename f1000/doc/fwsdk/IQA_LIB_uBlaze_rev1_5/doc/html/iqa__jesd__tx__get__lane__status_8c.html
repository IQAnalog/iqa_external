This API gets the status of the specified JESD204b TX lane.

The status bits in the returned value can be queried using the macros in the iqa_f1000_jesd.h header file.

NOTE: If the read operation fails, an error value will be returned and the status will be set to 0xDEADBEEF.

Parameters
f1000_id: An Integer that specifies the F1000 ID - Default 1
ep_idValue that represents the Element Processor ID.
Element Processor Value
A 0x1
B 0x2
laneLane number (0-7) of the TX lane to query.
pStatusPointer to the return uint32_t value.
Returns
: IQA_SUCCESS on Success else ERROR
