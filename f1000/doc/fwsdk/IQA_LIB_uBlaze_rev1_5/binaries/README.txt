IQA_LIB binaries README
=======================

This directory contains two executables that have been compiled for
64-bit Ubuntu.  The source for each these executables is also 
provided in the 'src' directory of this release.

The binaries are:

1.  fasm

    This is the Framer Assembler used to assemble the Merlin (Framer
    and Out-Of-Band) processor assembly code into machine code.

2.  merlinfw2ald

    This is a tool to create an Address-Length-Data (ALD) file containing
    multiple Merlin RX lane, TX lane and OOB binaries into a single
    packed .ald file for download to the F1000.
